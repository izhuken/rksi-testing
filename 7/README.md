## Рендер карточек

1. `card.js`

Файл содержит функцию `CardView`, которая возврящает строку в HTML формате карточки.

2. `tea.js`

Скрип рендерит карточки с чаем.

3. `dishes.js`

Скрип рендерит карточки с посудой для чаепития.
